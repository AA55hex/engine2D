Подробное описание

Класс-текстура Представляет собой абстракцию загруженой gl-текстуры.

Заметки
Координаты текстур нормализованы по диапазону [0;1]. Начало координатной плоскости (0;0) находится в левом нижнем углу окна.

Активировать буфер текстуры Активирует текстурный буфер по его номеру. Текстурный буфер используется для привязки текстуры к нему.
