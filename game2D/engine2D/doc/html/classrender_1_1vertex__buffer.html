Подробное описание

Буфер вершин Оболочка gl-буфера вершин. Представляет собой массив произвольных данных, реализация которых описывается вручную. Данный класс должен использоваться как хранилище входных данных рендера и использоваться совместно с vertex_array.

Заметки
Все взаимодействия с буфером вершин следует проводить опосредованно, с помощью vertex_array.
См. также
render::vertex_array
Заметки
Все динамические данные, поступающие в класс могут быть удалены после инициализации, так как они полностью копируются в память видеокарты.

Активация буфера вершин Привязывает буфер вершин к контексту, делая его активным. Одновременно активен только один буфер вершин.
