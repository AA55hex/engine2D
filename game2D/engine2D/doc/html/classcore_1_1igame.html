Подробное описание

igame - интерфейс взаимодействия с классом engine.

Данный интерфейс содержит весь необходимый функционал для взаимодействия с классом engine.

Заметки
Используйте реализацию данного класса в качестве класса-игры.

Действия по обработке изображения и звука. Перед вызовом данной функции из класса engine, происходит подготовка следущего игрового кадра. После вызова данной функции из класса engine, происходит обновление кадра.

Не производите запросы по обработке изображения вне диапазона данной функции.
