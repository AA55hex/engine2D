\doxysubsection{Подробное описание}
igame -\/ интерфейс взаимодействия с классом engine. 

Данный интерфейс содержит весь необходимый функционал для взаимодействия с классом engine. \begin{DoxyNote}{Заметки}
Используйте реализацию данного класса в качестве класса-\/игры. 
\end{DoxyNote}
