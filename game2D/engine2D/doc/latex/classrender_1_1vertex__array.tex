Массив атрибутов вершин (vao) Данный класс представляет собой оболочку вокруг соответствующей gl-\/структуры. Каждый vao-\/объект сохраняет информацию об активных атрибутах и вершинах, из которых будут браться данные для атрибутов. Разные vao имеют разные привязанные буферы и вершины и не зависят друг от друга

\doxysubsection{Подробное описание}
Массив атрибутов вершин (vao) Данный класс представляет собой оболочку вокруг соответствующей gl-\/структуры. Каждый vao-\/объект сохраняет информацию об активных атрибутах и вершинах, из которых будут браться данные для атрибутов. Разные vao имеют разные привязанные буферы и вершины и не зависят друг от друга 

\begin{DoxyNote}{Заметки}
Не работайте с вершинными буфермаи и буферами индексов без использования vertex\+\_\+array-\/объекта. Не используйте стандартный vao-\/объект для своих целей. 

При взаимподействии с \mbox{\hyperlink{classrender_1_1vertex__buffer}{render\+::vertex\+\_\+buffer}} всегда будьте уверены, что \mbox{\hyperlink{classrender_1_1vertex__buffer}{vertex\+\_\+buffer}} будет существовать до удаления vao-\/объекта 
\end{DoxyNote}
