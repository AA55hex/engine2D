\doxysubsection{Подробное описание}
Буфер вершин Оболочка gl-\/буфера вершин. Представляет собой массив произвольных данных, реализация которых описывается вручную. Данный класс должен использоваться как хранилище входных данных рендера и использоваться совместно с \mbox{\hyperlink{classrender_1_1vertex__array}{vertex\+\_\+array}}. 

\begin{DoxyNote}{Заметки}
Все взаимодействия с буфером вершин следует проводить опосредованно, с помощью \mbox{\hyperlink{classrender_1_1vertex__array}{vertex\+\_\+array}}. 
\end{DoxyNote}
\begin{DoxySeeAlso}{См. также}
\mbox{\hyperlink{classrender_1_1vertex__array}{render\+::vertex\+\_\+array}} 
\end{DoxySeeAlso}
\begin{DoxyNote}{Заметки}
Все динамические данные, поступающие в класс могут быть удалены после инициализации, так как они полностью копируются в память видеокарты. 
\end{DoxyNote}
