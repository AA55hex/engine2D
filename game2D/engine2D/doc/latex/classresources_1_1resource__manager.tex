\doxysubsection{Подробное описание}
Менеджер ресурсов Представляет собой абстракцию менеджера по загрузке и хранению подгружаемых ресурсов, таких, как текстуры, спрайты, звуки и т.\+д. Изначально содержит интерфейс по загрузке и хранению всех примитивов библиотеки. При необходимости, можно создать дочерний класс и дополнить его интерфейсом хранения и загрузки игровых объектов. 

\begin{DoxyNote}{Заметки}
Выводит логи в процессе создания/загрузки/возврата ресурса. 

Каождый объект имеет свой список ресурсов 
\end{DoxyNote}
